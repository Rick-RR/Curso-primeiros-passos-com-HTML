O que é HTML

(Voltar)

HTML é uma linguagem de marcação de textos para criação de paginas web. O html usa uma estrutura formada por conjuntos de elementos

que formam a pagina, esses elementos são chamados de tags. As tags são utilizadas para criação de titulos, paragrafos, enexação de imagem,

enexação de links, entre outros.

Estrutura Basica de um Html

(Voltar)

O html sempre inicia com a tag (<html>) para dizer ao navegador que ali começa o html

o html tem que ter as tags de cabeça e corpo (<head>, <body>) todas essas tags tem fechamento.

Bem, você provavelmente deve estar se perguntando "o que são tags?" não esquenta, esplicaremos logo, logo.

O que é a cabeça e o corpo do HTML

(Voltar)
Head (cabeça)

Na cabaça do html vão informações como por exemplo o titulo que aparecera em cima da aba da pagina.

Body (corpo)

Já o corpo é o que vai entrar em contato com o usuário, como por exemplo: Os titulos, paragrafos, imagem, etc.

O que são Tags

(Voltar)

As tags são utilizadas para dizer ao navegador o que está sendo construido. As tags tem que ter uma abertura e um fechamento, embora

algumas tags não precisem, pois elas mesmo ja se fecham como por exemplo a tag "<img/>".

A estrura basica de uma tag pode ser vista logo a baixo.

Ex. de tag para criação de um titulo:

<h1>Titulo da minha pagina</h1>

Perceba que eu abro a tag com "<h1>" para dizer ao navegador que inicia um titulo e fecho com "</h1>" para dizer ao navegador

que ali termina o titulo, note que a tag de fechamento precisa ter "/".

Tags Básicas

(Voltar)

<head>: Define a cabeça do documento

<body>: Define o corpo do documento

<h1 a h6>: Define o titulo, vai de 1 a 6 e quanto maior o número, menor a fonte do titulo.

<p>: Para paragrafos

<img>: Para adicionar imagem ao documento

<a>: Para adicionar um link

<strong>: Para deixar um texto em negrito

O que são Atributos

(Voltar)

Os atributos são palavras usadas dentro das tags de abertura para controlar o comportamento do elemento.

Ex. um atributo da tag:

(<img src="Link da imagem">)

Esse atributo "src" é usado para definir o link da sua imagem que vai aparecer na pagina.

Os atributos também podem ser usado para identificar um elemento.
